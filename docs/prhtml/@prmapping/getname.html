getname

GETNAME

Get mapping name

    NAME = GETNAME(W,N)

Input
 W Mapping
 N Number of characters in NAME (default: all)

Output
 NAME Mapping name

Description

If N given, the return string has exactly N characters. This is done by  truncation or by padding with blanks. This is useful for display purposes.
