ctranspose

CTRANSPOSE

Transpose mapping (sequential or affine)

     W = CTRANSPOSE(W)
     W = W'

Input
 W Mapping

Output
 W Transposed mapping

Description

For the case of affine (linear) mappings (or sequential combinations of  these), the transpose mapping is defined and returned. For other mappings  an error will be generated. See AFFINE.
