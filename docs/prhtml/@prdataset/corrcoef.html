corrcoef

CORRCOEF

Dataset overload

    D = CORRCOEF(A)

D is a dataset with the correlations between all features in A. The object  labels as well as the feature labels of D are equal to the feature labels  of A.

    D = CORRCOEF(A,B)

D is a dataset with the correlations between the 1D datasets A and B.
