getfeatlab

GETFEATLAB

Get feature labels of datafile

    FEATLAB = GETFEATLAB(A,STRING)

Input
 A Datafile
 STRING Indicate whether feature labels should be returned as strings  (if 'string' or 1), or not (default).

Output
 FEATLAB Feature labels

Description

Returns the labels of the features of datafile A. If STRING equals 'string'  or 1, the label list is converted to characters, which may be useful for  display purposes and annotation of graphs.
