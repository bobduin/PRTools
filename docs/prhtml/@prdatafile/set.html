SET

Set datafile fields

     A = SET(A,VARARGIN)

Sets datafile fields (given as strings in VARARGIN) of the datafile A.  E.G.: A = SET(A,'dir',DIR,'nlab',NLAB).  This is not different from using the field specific routines  (e.g. SETDIR(A,DIR)
