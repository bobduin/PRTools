getsize

GETSIZE

Dataset size and number of classes

   [M,K,C] = GETSIZE(A,DIM)

Input
 A Datafile
 DIM 1,2 or 3 : the number of the output argument to be returned

Output
 M Number of objects
 K Number of features
 C Number of classes

Description

Returns size of the datafile A and the number of classes. C is determined  from the number of labels stored in A.LABLIST. If DIM = 1,2 or 3, just  one of these numbers is returned, e.g. C = GETSIZE(A,3).  The feature size K is determined by reading and processing the first  object of A.
