IM_ROTATE

Fixed mapping for image rotation

   B = IM_ROTATE(A,ALF)
   B = A*IM_ROTATE([],ALF)
   B = A*IM_ROTATE(ALF)

Input
 A Dataset with object images (possibly multi-band)
 ALF Rotation angle (in radians),  default: rotation to main axis

Output
 B Dataset with rotated object images

Description

The objects stored as images in the dataset or datafile A are rotated  using the IMROTATE command.
