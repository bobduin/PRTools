ONESTM

Trainable mapping, maps any object on ones

    W = ONESTM(A)
    W = A*ONESTM

Input
 A Array or dataset

Output
 W Ones mapping, if applied to dataset, it returns a dataset with ones  of the same size.

Description

This is a trainable ones mapping that maps any dataset on ones.  The difference with the fixed mapping is ONESM is that ONESM(A) returns  A and that W = ONESTM(A) returns a mapping with input and output  dimensionality the feature size of A. Consequently B*W returns ones with  the size of and checks the feature size of B against A.

This routine facilitates the construct of dyadic trainable mappings,  like W = (FISHERC < 0.5)
