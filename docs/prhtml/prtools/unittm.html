unittm

UNITTM

Trainable unit mapping

    W = UNITTM(A)
    W = A*UNITTM

Input
 A Array or dataset

Output
 W Unit mapping, if applied to dataset, it is returned unchanged

Description

This is a trainable unit mapping that maps any dataset on itself.  The difference with the fixed mapping is UNITM is that UNITM(A) returns  A and that W = UNITTM(A) returns a mapping with input and output  dimensionality the feature size of A. Consequently B*W returns B, but  checks the feature size against A.
