PREX_SPATM

PRTools example on spatial smoothing of image classification

Uses a multi-band image which is first segmented (classified) into 3 classes based on the EM-clustering applied to a feature space  defined by the bands. The result is then smoothed by using the spatial  information on the neighboring pixels.
