IM_PROFILE

Fixed mapping computating horizontal and vertical image profiles

   P = IM_PROFILE(A,NX,NY)
   P = A*IM_PROFILE([],NX,NY)
   P = A*IM_PROFILE(NX,NY)

Input
 A Dataset with object images dataset (possibly multi-band)
 NX Number of bins for horizontal profile
 NY Number of bins for vertical profile

Output
 P Dataset with profiles replacing images (possibly multi-band)

Description

Computes for the images in A a 1*(NX+NY) vector P storing the horizontal and  vertical profile (i.e. normalized image projections) given by NX respectively  NY points. Because of the normalisation holds sum(P) = 1.  The computation is based on a bilinear interpolation of A to an image of NX*NY points. Consequently is the horizontal profile (P(1:NX)) slightly dependent  on NY and the vertical profile (P(NX+1:NX+NY)) slightly dependent on NX.  Defaults : NY = NX, NX = 16.
