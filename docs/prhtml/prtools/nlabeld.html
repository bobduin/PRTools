nlabeld

NLABELD

Return numeric labels of classified dataset (c

    NLABELS = NLABELD(Z)
    NLABELS = Z*NLABELD
     NLABELS = NLABELD(A,W)
    NLABELS = A*W*NLABELD

Input
 Z Classified dataset, or
 A,W Dataset and classifier mapping

Output
 NLABELS Column vector of numeric labels)

Description

Returns the numberic labels of the classified dataset Z (typically the  result of a mapping or classification A*W). For each object in Z (i.e.  each row) the feature label or class label (i.e. the column label) of the  maximum column value is returned. This corresponds with the classes  stored in W, which can be found by GETLABELS(W).
