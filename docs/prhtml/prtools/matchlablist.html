MATCHLABLIST

Match entries of lablist1 with lablist2

     I = MATCHLABLIST(LABLIST1,LABLIST2)

Input
 LABLIST1 list of class names
 LABLIST2 list of class names

Output
 I indices for LABLIST1 appearing in LABLIST2

Description

Find the indices of places where the entries of LABLIST1 appear  in LABLIST2, i.e. LABLIST1 = LABLIST2(I).  Note that this operation is not symmetric, changing the order of  LABLIST1 and LABLIST2 changes I!  I(i) = 0 for labels appearing in LABLIST1 that are not in LABLIST2.
