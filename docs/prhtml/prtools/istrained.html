ISTRAINED

Test on trained mapping

     I = ISTRAINED(W)
     ISTRAINED(W)

True if the mapping type of W is 'trained' (see HELP MAPPINGS). If  called without an output argument ISTRAINED generates an error if the  mapping type of W is not 'trained'.
