use_old_mapping_times

USE_OLD_MAPPING_TIMES

     USE_OLD_MAPPING_TIMES(FLAG)

Input
 FLAG TRUE or FALSE

Description

For FLAG = TRUE (1) the old version of the multiplication of mappings  (e.g. W = 100*FISHERC) will be used. This may be reset by a call with  FLAG = FALSE
