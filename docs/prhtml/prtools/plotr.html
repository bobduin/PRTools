plotr

PLOTR

Plot regression

     PLOTR(W)
     PLOTR(W,CLR)

Plot the regression function W, optionally using plot string CLR.  This plot string can be anything that is defined in plot.m.  For the best results (concerning the definition of the axis for  instance) it is wise to first scatter the regression data using  SCATTERR(A).

The resolution of the plot is determined by the global parameter  GRIDSIZE.
