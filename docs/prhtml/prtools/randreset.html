randreset

RANDRESET

Reset state of random generators

RANDRESET - reset states to 1 RANDRESET(STATE) - reset states to STATE STATE = RANDRESET - retrieve present state (no reset)  OLDSTATE = RANDRESET(STATE) - retrieve present state and reset
