ISSTACKED

Test on stacked mapping

    N = ISSTACKED(W)
    ISSTACKED(W)

Input
 W Mapping

Output
 N Scalar, 1 if W is a stacked mapping, 0 otherwise

Description

Returns 1 for stacked mappings. If no output is requested, false outputs  are turned into errors. This may be used for assertion.
