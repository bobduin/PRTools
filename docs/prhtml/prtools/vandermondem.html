VANDERMONDEM

Fixedd mapping, extend data matrix

     Z = VANDERMONDEM(X,N)
     Z = X*VANDERMONDEM([],N)
     Z = X*VANDERMONDEM(N)

Input
 X Data matrix
 N Order of the polynomail

Output
 Z New data matrix containing X upto order N

Description

Construct the Vandermonde matrix Z from the original data matrix X by  including all orders upto N. Note that also order 0 is added
Z = [ones X X^2 X^3 ... X^N]  This construction allows for the trivial extension of linear methods  to obtain polynomail regressions.
