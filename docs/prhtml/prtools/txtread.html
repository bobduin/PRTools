TXTREAD

Read text file

    A = TXTREAD(FILENAME,N,START)

Input
 FILENAME Name of delimited ASCII file
 N Number of elements to be read (default all)
 START First element to be read (default 1)

Output
 A String

Description

Reads the total file as text string into A
