onec

ONEC

Single class classifier

    W = ONEC(A)
    W = A*ONEC

Input
 A Dataset

Output
 W Trained classifier

Description

This classifier serves as an escape to irregular situations, e.g. when a  dataset contains objects of a single class. It creates a trained  classifier that assigns all objects to a one class. In case the priors  of A are set this will be the class with the maximum prior probability  times the classsizes. In case the priors of A are not set objects to be  classified will be assigned to the most frequent class in A.
