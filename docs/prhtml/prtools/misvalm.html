MISVALM

Trainable mapping handling the missing values in a dataset

     W = MISVAL(A,TYPE,K)
     W = A*MISVAL(TYPE,K)
     B = A*W

Input
 A Dataset, containing NaNs (missing values), to be used for training
 TYPE Cell array with strings mappings, or values defining a  substitution, one cell per feature. Default: on cell per feature.
 K Vector with indices of features to be handled. Default: all

Output
 B Dataset with missing values corrected
 W Trained mapping used to execute new datasets

Description

The following values for TYPE are possible
'remove' *remove objects (rows) that contain missing values (default)  'f-remove' remove features (columns) that contain missing values  'mean' fill the entries with the mean of the features  'median' fill the entries with the median of the features  'majority' fill the entries with the majority of the features  (typically usefull for categorical features).  *fill the entries with a fixed constant  fixed(*), trained(*) or untrained regression mapping.

The indicated options (*) just store them in W without inspecting A.  The f-remove option stores in W the features of A having missing values.  They are thereby removed from B.

Note that replacing missing values by feature means (medians or  majorities) is a very simple solution that might be far from optimal.  The values computed for a training set A are applied to the dataset B.

In case TYPE{N} is a mapping, it should be specific for feature N, and it  will be used as B(L,N) = B(L,RN)*TYPE(N), in which RN is the list of all  feature indices except N and L is the list of all objects for which  feature N is missing.

If TYPE{N} is untrained it will be trained by all objects in A for which  just feature N is missing. For datasets with many missing values use  FEATSEL(K)*TYPE{N} instead of TYPE{N} in which K is a set of features for  which missing values are sparse. Training is then restricted to the  features K.
