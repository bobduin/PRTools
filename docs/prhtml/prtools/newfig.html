newfig

NEWFIG

Create new figure on given position

   NEWFIG(FIGURE_NUMBER,FIGURE_PER_ROW)

Input
 FIGURE_NUMBER Number of the figure
 FIGURE_PER_ROW Figures per row (default: 4)

Output

Description

Creates figure number FIGURE_NUMBER and places it on the screen,  such that (when sufficient figures are created), the screen is  covered by an array of figures, with FIGURE_PER_ROW figures per row.
