im_invert

IM_INVERT

Fixed mapping for images inversion

  A = IM_INVERT(A)
  A = A*IM_INVERT

Description

Inverts image A by subtracting it from its maximum. Note that binary  images can better be inverted by A = ~A. In that case also A = 1-A can  be done.
