onesm

ONESM

Fixed mapping, maps any object on ones

    W = ONESM
    B = A*ONESM
    B = ONESM(A)

Input
 A Array or dataset

Output
 W Ones mapping, maps any object on ones(1,size(A,2))
 B ones(size(A))

Description

This is a fixed mapping that maps any dataset on ones. There is  also a trainable ons mapping, named ONESTM.
