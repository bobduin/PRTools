MAPPINGTOOLS

Macro defining some mappings:

   ARGMIN, ARGMIN1, ARGMIN2, ARGMAX, ARGMAX1, ARGMAX2
   EXPM, NEXPM,
   ARGMAXMAX, ARGMAXMIN, ARGMINMAX, ARGMINMIN, ARGMEANMAX, ARGMEANMIN
   MINK1, MINK2

This is not a function. Calling MAPPINGTOOLS just loads the above  mappings ready to use. They can be called as J = A*ARGMIN in which A is a  dataset or an array of doubles. It returns a vector of object indices J pointing to the feature (column) with the lowest value.
