PRLOAD

Similar to LOAD matfile, but converts old datasets, datafiles, mappings

   PRLOAD FILE
   PRLOAD(FILE,VARARGIN)
   S = PRLOAD(FILE,VARARGIN)

PRLOAD behaves similar to LOAD, but converts PRTools4 variables to  PRTools5.
