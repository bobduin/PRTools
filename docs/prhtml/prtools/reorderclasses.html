REORDERCLASSES

Reorder the lablist

      X = REORDERCLASSES(X,LABLIST)
      X = REORDERCLASSES(X,I)

Input
 X (labeled) dataset
 LABLIST correctly ordered lablist
 I permutation vector

Output
 X dataset with reordered classes

Description

Change the order of the classes in dataset X to the one defined in  LABLIST. When only a single class label is given in LABLIST, this  class is put in the first position, and the others are just kept in  the order they already were.

For example, when I have a dataset A with a labellist {'apple'  'banana' 'pear'} and I perform B=REORDERCLASS(A,3) (or identically  B=REORDERCLASS(A,[3 1 2])), I will get a dataset B with a new  labellist of {'pear' 'apple' 'banana'}.
