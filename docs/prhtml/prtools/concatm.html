concatm

CONCATM

Fixed-cell mapping concatenating cell array of datasets or mappings

    W = V*CONCATM
    B = A*CONCATM

Input
 V Cell array of mappings
 A Cell array of datasets

Output
 W Concatenation of mappings: W = [V{:}]
 A Concatenation of datasets: B = [A{:}]

Description

This routine is an exception of the general treatment of cell arrays and  mappings in PRTools. As a rule the elements of a cell array are processed  one by one by a mapping. This routine takes the entire cell array and  concatenates its elements.
